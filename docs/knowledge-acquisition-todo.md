## 公众号与科普网页

公众号文章可以补，但建议先做“索引”，不要直接全文复制进公开仓库。

每篇至少记录：

- 标题
- 公众号/机构名
- 作者或审核者
- 发布日期
- URL 或截图来源
- 主题标签：发热、PICC、饮食、心理、化疗解释等
- 是否官方机构/医院/公益组织
- 是否获得转载或公开使用授权
- 适合改写成哪条 knowledge

优先收集类型：

- 儿童血液肿瘤专科医院或科室官方公众号科普。
- 儿童肿瘤基金会、公益组织、患者教育平台文章。
- 护理团队发布的 PICC、输液港、口腔护理、发热就医宣教。
- 儿童心理、住院安抚、兄弟姐妹沟通相关科普。

不建议收集：

- 未署名、无审核者、来源不明的治疗方案文章。
- 带强烈药物推荐、偏方、保健品导向的内容。
- 需要付费授权但无法确认版权的全文材料。

## 入库前检查

- 医学内容是否来自权威指南、官方患者教育或医生审核资料。
- 是否只做原创摘要，未复制长段原文。
- 是否避免治疗方案、药物剂量、停药换药建议。
- 是否标记 `draft_doctor_review_required`。
- 是否能关联到 `source_refs` 和 `asset_refs`。
- 是否不含真实家庭隐私、联系方式、病历号或检查单号。
