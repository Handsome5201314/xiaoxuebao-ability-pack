# 返校和日常生活准备

孩子什么时候能返校或参加集体活动，需要结合治疗阶段、血细胞情况、感染风险、体力和医生建议。不要只按孩子想不想去来决定。

家长可以提前和学校沟通：孩子可能需要减少感染暴露、避免剧烈碰撞、按时复诊、必要时休息，以及保护隐私和减少标签化。

如果孩子正在发热、明显乏力、血细胞很低或医生提示需要保护性隔离，应先按医疗团队建议安排。
